## Architecture

The Instagram service (port 3005) now serves as a unified API combining:
- Comment operations (native, via InstagramDriver)
- DM operations (proxy to port 3100 DM service, with fallback)
- Profile operations (via Safari JS)
- Session management (in-memory)
- AI features (via InstagramAICommentGenerator)
- Rate limiting and auth (middleware layer)

## Key Decisions
- Rate limit: 200 req/min by default (configurable via RATE_LIMIT_MAX)
- Auth token: configurable via INSTAGRAM_API_TOKEN env var
- DM proxy: forwards to port 3100, returns 503 if unavailable
- Session: in-memory (no persistence across restarts)
- AI scoring: local analysis engine (no API key required)
